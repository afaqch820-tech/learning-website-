STUDENT GUIDE · WEB DEVELOPMENT

ویب سائٹ کیسے بنائیں؟

یہ ایک مکمل، آسان اور قدم بہ قدم گائیڈ ہے جو ایک طالب علم کو صفر سے اپنی پہلی ویب سائٹ بنانا سکھاتی ہے۔ نیچے دیے گئے مراحل کو ترتیب سے پڑھیں اور خود عملی طور پر آزمائیں۔

HTML CSS JavaScript Hosting
STEP 01

منصوبہ بندی کریں

کوڈ لکھنے سے پہلے یہ طے کریں کہ آپ کی ویب سائٹ کس بارے میں ہے اور اس میں کون کون سے صفحات ہوں گے۔

مقصد طے کریں

سوچیں کہ ویب سائٹ کس کے لیے ہے — ذاتی تعارف، پورٹ فولیو، بلاگ یا کوئی چھوٹا پراجیکٹ؟

صفحات کی فہرست بنائیں

مثلاً: ہوم پیج، میرے بارے میں، رابطہ — کاغذ پر ایک سادہ خاکہ بنا لیں۔

STEP 02

HTML سے ڈھانچہ بنائیں

HTML ویب سائٹ کی بنیاد ہے۔ یہ صفحے کا ڈھانچہ اور مواد بتاتی ہے — جیسے عمارت کی اینٹیں۔

<!-- یہ ایک بنیادی HTML صفحہ ہے -->
<!DOCTYPE html>
<html>
  <head>
    <title>میرا پہلا صفحہ</title>
  </head>
  <body>
    <h1>خوش آمدید</h1>
    <p>یہ میری پہلی ویب سائٹ ہے۔</p>
  </body>
</html>

ہر <tag> ایک مخصوص کام کرتا ہے: h1 بڑا عنوان، p عام تحریر (پیراگراف)۔

STEP 03

CSS سے خوبصورت بنائیں

CSS رنگ، فونٹ، اور ترتیب سنبھالتی ہے — یہ ویب سائٹ کا لباس اور سجاوٹ ہے۔

h1 {
  color: darkblue;
  font-size: 40px;
  text-align: center;
}
STEP 04

خود آزمائیں — لائیو ایڈیٹر

نیچے دیے گئے خانے میں کوڈ تبدیل کریں اور دائیں طرف فوری نتیجہ دیکھیں۔ یہی سیکھنے کا بہترین طریقہ ہے۔

لائیو ایچ ٹی ایم ایل لیب

LIVE

ترکیب: رنگ کا کوڈ بدل کر دیکھیں، جیسے color:red یا color:green — نتیجہ فوراً بدل جائے گا۔

STEP 05

JavaScript سے حرکت پیدا کریں

JavaScript ویب سائٹ کو ذہین بناتی ہے — بٹن دبانے پر کچھ ہونا، فارم چیک کرنا، وغیرہ۔

<button onclick="alert('شکریہ!')">
  دبائیں
</button>
STEP 06

ضروری اوزار

شروع کرنے کے لیے آپ کو صرف یہ چاہیے:

  • کوڈ ایڈیٹرمثلاً VS Code — جہاں آپ کوڈ لکھیں گے۔
  • ویب براؤزرمثلاً Chrome — جہاں نتیجہ دیکھیں گے۔
  • فولڈراپنی فائلوں (index.html, style.css) کو ایک جگہ رکھیں۔
  • حوصلہغلطیاں ہونا معمول ہے — مشق سے سیکھیں۔
STEP 07

ویب سائٹ کو انٹرنیٹ پر شائع کریں

جب صفحہ تیار ہو جائے تو اسے دنیا کے سامنے لانے کے لیے کسی مفت ہوسٹنگ سروس کا استعمال کریں، جیسے GitHub Pages یا Netlify۔ اپنی فائلیں اپلوڈ کریں اور آپ کو ایک لنک مل جائے گا جسے کوئی بھی کھول سکتا ہے۔

ہر بڑی ویب سائٹ ایک چھوٹے سے <h1> سے شروع ہوئی تھی
